Photonic

Photonic is a game about a light particle trying to get to the core of a particle accelerator.
Made for the Global Game Jam 2017 with a team of game development enthusiasts in Brazil.
It was the first game that we ever did. My main role was level designer and 3D artist.
The game was made using Unreal Engine and Autodesk Maya.
Available for Windows

Find it on: Github or GGJ website

Mission, Transmission

A game about logic and 80's synth. Be smart, avoid the NSA and get your DataDrone throught. We're counting on you. Hack the Planet!.
Made for the Global Game Jam 2018 with a team of game development enthusiasts in Brazil.
My second jam game, working with a big team. My main role was lead designer, 3D artist and level designer.
The game was made using Unity Engine and Autodesk Maya.
Available for Windows

Find it on: Github or GGJ website

Dodge the Circle

A rhythm game for Android, made as final project for the Android Development course at CICCC.
Dodge the outgoing barriers and see how long you can survive.
My main role was lead designer and developer.
The game was made using Unity Engine and 2D assets.

Find it on: Github
